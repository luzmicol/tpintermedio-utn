TP Intermedio - Front End Developer

Descripción:
Recreación de una parte de la página principal de Nave Negocios utilizando HTML y CSS.
El objetivo fue practicar maquetación web, diseño responsive y buenas prácticas de desarrollo frontend.

Sitio de referencia:
https://navenegocios.ar/home

Tecnologías utilizadas:
HTML5
CSS3
Flexbox

Objetivos:
Crear una interfaz responsive.
Aplicar HTML semántico.
Utilizar Flexbox para la distribución de elementos.
Mantener un código limpio y organizado.
